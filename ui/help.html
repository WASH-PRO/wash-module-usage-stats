Usage Stats Collector

The module loads records from GET /api/crm/usage-stats and aggregates usage seconds and launch counts per car wash for the selected period.

Settings

  • poll_interval — poll interval (sec).
  • lookback_hours — lookback window in hours (1–168).

Data

Result in data/last_snapshot.json: fields byWash, totalUsageTimeSec, totalLaunchCount.
